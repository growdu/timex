## 1. 简介

时光机器是一款面向个人成长记录与人生回忆沉淀的多模态时间档案产品。它不只是日记，也不只是相册 -- 它将你人生中的**事件、人物、地点、瞬间**串联成一张多维网络，让你从**时间、空间、感情、事业、亲情、朋友**六个维度回顾和整理自己的人生。

### 核心理念

| 概念 | 说明 |
|------|------|
| **事件优先** | 以事件为核心单元，而非散乱的素材堆砌 |
| **六条线** | 时间 · 空间 · 感情 · 事业 · 亲情 · 朋友，多维导航 |
| **回忆录** | 将事件编排成章节，产出可打印的故事书 |
| **隐私买断** | 一次性买断，数据自主可控 |

## 3. 统计大屏

登录后默认进入**统计大屏**（点击导航栏「大屏」可随时回到此页）。

### 3.1 大屏内容

大屏以卡片式布局展示你的时光数据总览：

| 卡片 | 内容 |
|------|------|
| **统计概览** | 事件总数、人物总数、地点总数、瞬间总数、回忆录总数 |
| **时间线分布** | 按年份统计事件数量（条形图） |
| **阶段分布** | 按人生阶段统计（学生/初入职场/创作试验/家庭时刻/自定义） |
| **最近事件** | 最近 6 条事件，可直接点击查看详情 |
| **地点分布** | 事件最多的 10 个地点 |
| **核心人物** | 出场次数最多的人物排行 |
| **素材构成** | 瞬间类型分布（照片/视频/音频/文字） |
| **回忆录** | 已创建的回忆录列表及章节数 |

### 3.2 导出入口

大屏底部有**导出与打印**区域，提供三种导出方式：

- **📸 导出相册** -- 按事件展示的时光相册
- **📅 导出时间线** -- 完整事件按年分组
- **📖 导出故事书** -- 选择回忆录，导出章节故事书

详见 [第 9 节：导出与打印](#9-导出与打印)。

## 4. 记录事件（时间线）

点击导航栏 **「时间线」** 进入事件时间线页面。

### 4.1 浏览时间线

- 事件按**年份**倒序排列
- 左侧栏可按**年份**和**人生阶段**筛选
- 每个事件卡片显示：标题、日期、地点、摘要、关联人物、素材数量

### 4.2 添加事件

1. 在时间线页面，点击 **「添加事件」** 按钮
2. 填写事件信息：
   - **标题**（必填）：简短描述，如「女儿出生」「入职大厂」
   - **日期**（必填）：事件发生日期
   - **地点**：选择已有关联地点，或输入新地点
   - **阶段**：学生时代 / 初入职场 / 创作试验 / 家庭时刻 / 自定义
   - **摘要**：一两句话概括
   - **详细描述**：完整记录事件经过
   - **关联人物**：选择与此事件相关的人物
3. 点击 **「保存」**

### 4.3 编辑与删除

- 点击事件卡片进入详情页
- 在详情页可编辑或删除事件

### 4.4 事件详情页

每个事件有独立的详情页（`/events/:id`），展示：
- 事件完整信息
- 关联的瞬间（照片、视频等）
- 关联的人物列表
- 关联的地点信息

## 5. 空间地图

点击导航栏 **「空间」** 进入空间地图页面。

### 5.1 功能

- 在地图上标记你所有去过的地方
- 每个地点显示关联的事件数量
- 点击地点可查看该地点的所有事件
- 支持地点详情页（`/places/:id`），展示地点档案

### 5.2 添加地点

1. 在空间页面点击 **「添加地点」**
2. 填写：名称、类型（城市/旅行/家庭/日常）、简介、经纬度
3. 保存后地点会出现在地图上

## 6. 人物关系

点击导航栏 **「人物」** 进入人物管理页面。

### 6.1 功能

- 展示所有人物卡片（头像、姓名、角色、简介）
- 每个人物显示关联的事件数量
- 点击人物进入详情页（`/people/:id/detail`），展示该人物参与的所有事件

### 6.2 添加人物

1. 点击 **「添加人物」**
2. 填写：姓名、角色（如「家人」「同事」「朋友」）、简介
3. 保存

### 6.3 人物与事件关联

在添加或编辑事件时，可以选择关联人物。这样就能从人物维度查看共同经历。

## 7. 回忆录编辑器

点击导航栏 **「回忆录」** 进入回忆录编辑器。

### 7.1 创建回忆录

1. 点击 **「新建回忆录」**
2. 填写标题和简介
3. 保存后进入编辑器

### 7.2 编辑章节

回忆录编辑器采用**三栏布局**：

| 栏目 | 功能 |
|------|------|
| **左侧：章节树** | 管理 chapters，可添加、删除、排序、点击切换 |
| **中间：正文区** | 编辑当前章节的标题和正文内容 |
| **右侧：来源库** | 展示与当前章节关联的事件，可从中选取素材 |

### 7.3 关联事件到章节

1. 在右侧来源库中选择事件
2. 将事件关联到当前章节
3. 章节正文可引用事件内容

### 7.4 分享回忆录

- 点击 **「分享」** 可生成分享链接
- 拥有链接的人可以查看回忆录内容

## 8. 添加数据

### 8.1 数据模型

时光机器的数据由四种核心实体构成：

```
事件 (Event) ──┬── 关联 ──→ 人物 (Person)
               ├── 归属 ──→ 地点 (Place)
               └── 包含 ──→ 瞬间 (Moment)

回忆录 (Memoir) ── 包含 ──→ 章节 (Chapter) ── 关联 ──→ 事件 (Event)
```

### 8.2 瞬间（素材）

瞬间是附着在事件上的多媒体素材：

| 类型 | 说明 |
|------|------|
| 📷 照片 | 图片素材 |
| 🎬 视频 | 视频素材 |
| 🎵 音频 | 音频素材 |
| 📝 文字 | 文字记录 |

**添加方式**：在事件详情页点击「添加瞬间」，选择类型并填写内容。

### 8.3 AI 智能增强

系统支持 AI 辅助功能（需配置 AI 服务）：
- **图片标注**：自动识别图片内容并生成标签
- **图片摘要**：自动为图片生成文字描述
- **音频转写**：将音频转录为文字
- **事件摘要**：自动生成事件摘要
- **章节摘要**：自动总结章节内容

## 9. 导出与打印

### 9.1 时光相册

**入口**：大屏 → 导出与打印 → 📸 导出相册

**功能**：
- 按时间顺序展示所有事件
- 每个事件显示：日期、标题、地点、摘要、人物、瞬间素材
- 可按年份筛选
- 点击 **「🖨 打印 / 保存PDF」** 调用浏览器打印

**打印技巧**：
- 在打印对话框中选择「保存为 PDF」
- 建议选择纵向打印
- 纸张大小推荐 A4

### 9.2 完整时间线

**入口**：大屏 → 导出与打印 → 📅 导出时间线

**功能**：
- 按年份分组展示全部事件
- 每个事件显示：日期、标题、地点、人物、素材数量
- 适合作为年度大事记打印

### 9.3 故事书

**入口**：大屏 → 导出与打印 → 📖 导出故事书 → 选择回忆录

**功能**：
- 以回忆录章节为结构，展示完整故事
- 每章显示：章节标题、正文内容、关联事件
- 适合作为个人回忆录印刷输出

### 9.4 打印参数建议

| 参数 | 推荐值 |
|------|--------|
| 纸张 | A4 |
| 方向 | 纵向 |
| 边距 | 默认 |
| 缩放 | 100% |
| 背景图形 | 关闭（节省墨水） |

## 10. 制作美好回忆的技巧

### 10.1 记录原则

1. **事件导向**：以「发生了什么」为单元，而非「今天的心情」。一个事件 = 一个有起止的故事。
2. **具体而非抽象**：写「女儿第一次叫妈妈」而非「家庭生活」。具体的标题更容易回忆。
3. **关联人物和地点**：给每个事件关联相关的人和地点，这样从人物或地图维度也能找到这段记忆。
4. **定期整理**：建议每周或每月回顾一次，把零散的瞬间归入事件，把事件编入回忆录。

### 10.2 素材管理

1. **照片配文字**：上传照片时，写一两句当时的情况和感受。十年后看回来，文字比照片更有温度。
2. **视频抓重点**：视频素材标注关键时间点或内容摘要。
3. **音频转录**：重要对话或会议录音可以用 AI 转写为文字。
4. **混合记录**：一个事件可以同时有照片、视频、文字 -- 多维度记录让回忆更立体。

### 10.3 回忆录写作

1. **以主题分章**：如「北漂三年」「西藏之旅」「小雨的诞生」，每个主题一章。
2. **先列大纲**：先把章节标题列出来，再逐章填充内容。
3. **引用事件**：把相关事件关联到章节，让故事有据可查。
4. **定期回顾**：回忆录是活文档，可以不断修订和补充。

### 10.4 六条线维度

利用六条线维度来发现你人生中被忽略的部分：

| 维度 | 回顾方法 |
|------|----------|
| ⏱ 时间 | 按年回顾，看哪些年份事件密集，哪些年份空白 |
| 🗺 空间 | 打开地图，看你去过哪些地方，还有哪里没去 |
| ❤ 感情 | 从人物角度，看谁在你生命中出场最多 |
| 💼 事业 | 按阶段筛选，看职业发展的转折点 |
| 🏡 亲情 | 筛选家庭时刻，回顾家人共同经历 |
| 👥 朋友 | 查看朋友相关事件，维系重要友谊 |

### 10.5 导出成册

1. **年度相册**：每年年底，导出该年度的相册，打印成实体册
2. **人生故事书**：把回忆录导出为故事书，打印装订，作为传家之物
3. **时间线大事记**：导出完整时间线，作为个人编年史
4. **送礼**：把回忆录分享链接发给亲友，或打印成册作为礼物

## 11. 常见问题

### Q: 忘记密码怎么办？
A: 目前请联系管理员重置密码。

### Q: 试用到期后数据会丢失吗？
A: 不会。试用到期后账号会被限制功能，但数据保留。购买 License 后即可恢复全部功能。

### Q: 如何购买 License？
A: 在「License」页面输入授权码激活。支持年费（¥99/年）和买断（¥399起）。

### Q: 数据存储在哪里？
A: 数据存储在你部署的服务器上（PostgreSQL 数据库 + 对象存储），你完全拥有数据主权。

### Q: 导出的 PDF 格式不好看？
A: 打印时建议：选择 A4 纸张、纵向、关闭背景图形。不同浏览器打印效果略有差异，推荐 Chrome。

### Q: 可以多人使用吗？
A: 每个用户有独立的数据空间，互不可见。可以注册多个账号分别使用。

### Q: AI 功能需要什么配置？
A: AI 功能支持 OpenAI 兼容接口（包括 DeepSeek、Azure 等）和本地 Ollama。在服务端 `.env` 中配置 `OPENAI_API_KEY` 或 `OLLAMA_HOST` 即可。未配置时自动降级为 Mock 模式。
